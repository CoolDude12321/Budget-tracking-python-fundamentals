# Budget-tracking-python-fundamentals
Personal Expense Tracker - A menu-driven Python application to log expenses, categorize spending, set monthly budgets, & save/load data using CSV file handling.

## Features
- Add daily expenses with category & amount
- View all recorded expenses
- Categorize Spending (Food, Travelling, etc.)
- Set & track monthly budget
- Save expene data to a csv file
- Load previous data from the csv file

## How it Works
1. User interacts with a menu-driven console interface
2. Expense data is stored using Python lists & dictionaries
3. csv files are used for persistent data storage
4. Budget limits are checked against total expenses

## Files
- Expense Tracker.ipynb : Main python code
- Problem Statement.pdf : Project problem description
- README.md : Project documentation
